# **Dependencies**
```
sway, rofi, mako, fastfetch, waybar, swaybg, hyprshot/flameshot, kitty, thunar, ttf-jetbrains-mono, otf-font-awesome, cliphist, wiremix
```

# **Installation**
```
git clone https://github.com/wirosmango/sway-dotfiles.git
cd sway-dotfiles
cp -r * ~/.config
```

**Done!**
